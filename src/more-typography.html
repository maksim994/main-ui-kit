Заголовки

H1 — Основной заголовок страницы

Поддерживающий абзац для H1. Этот текст помогает оценить отступы под основным заголовком и соотношение между крупными заголовками и основным текстом.

H2 — Заголовок раздела

Заголовок раздела используется для деления больших областей контента на смысловые группы. Он должен быть выразительным, но не перегружать страницу.

H3 — Заголовок подраздела

H3 хорошо подходит внутри карточек, контентных блоков, групп функций и разделов форм.

H4 — Заголовок небольшого блока

Этот уровень подходит для небольших областей контента и компактных информационных групп.

H5 — Второстепенный заголовок

Небольшие заголовки можно использовать внутри виджетов, боковых панелей и структурированных метаданных.

H6 — Минимальный заголовок

Это самый маленький уровень заголовка. Используйте его только если структура действительно требует этого.

Размеры текста

Текст XXL — пример 22px для выразительного вспомогательного текста.

Текст XL — пример 20px для большого абзаца или вводного текста.

Текст LG — пример 18px для крупного основного текста.

Стандартный основной текст — 16px. Это базовый размер абзаца для большинства интерфейсов.

Текст SM — пример 14px для второстепенного интерфейсного текста, подписей и подсказок.

Текст XS — пример 12px для подписей, метаданных и второстепенных описаний.

Пример подписи с использованием класса caption для дополнительной информации.

Цвета текста

Основной текст — используется для главного контента и ключевых элементов интерфейса.

Второстепенный текст — используется для описаний и менее заметного контента.

Приглушённый текст — используется для метаданных, вспомогательной информации или отметок времени.

Фирменный текст — используется для выделенного текста и ссылок.

Текст успеха — используется для положительных сообщений о статусе.

Текст предупреждения — используется для предупреждающей информации.

Текст ошибки — используется для сообщений валидации и критических уведомлений.

Информационный текст — используется для нейтральных информационных сообщений.

Абзацы и ритм

Хорошая типографика — это не только размер шрифта. Это также ритм, межстрочный интервал, отступы, контраст и иерархия. Этот абзац используется для проверки удобства чтения внутри обычного контентного блока.

Второй абзац позволяет оценить вертикальные отступы между текстовыми блоками. Текст должен восприниматься спокойно, легко читаться и выглядеть сбалансированно как на десктопе, так и на мобильных устройствах.

Эту область также можно протестировать с помощью утилиты .text-reset, если необходимо контролировать отступы абзацев внутри CMS-контента.
